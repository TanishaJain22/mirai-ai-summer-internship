# 🎨 AI Image Creator Studio

An AI-powered image generator built with **Streamlit** and the **Pollinations.ai** API. Type a prompt, pick a style, and generate stunning images instantly.

---

## Features

- **Art Style Selector** — Choose from 8 styles: Photorealistic, 3D Render, Pixel, Ghibli, Sketch, Manga, Anime, Cartoon
- **Width & Height Sliders** — Control output resolution (256–1024 px)
- **✨ Magic Enhance** — Silently boosts prompt quality with pro-level keywords for better results
- **🎲 Surprise Me!** — Generates a random creative prompt for instant inspiration
- **Download Button** — Save the image as a `.png` with a style-based filename

---

## How to Run Locally

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the app
streamlit run app.py
```

The app will open at `http://localhost:8501`.
