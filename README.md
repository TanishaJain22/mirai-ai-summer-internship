# Mirai AI Summer Internship

AI Summer Internship at **Mirai School of Technology** (AI Builder Track)

---

## About

This repo contains hands-on assignments completed during the internship, covering applied AI, NLP, and generative AI using the Gemini API and Streamlit.

---

## Projects

| # | Folder | Description |
|---|--------|-------------|
| 1 | [`identity-echo-interface`](./identity-echo-interface) | An interactive Streamlit app with conditional input validation and a token cost estimator |
| 2 | [`gemini-persona-chatbot`](./gemini-persona-chatbot) | A Streamlit chatbot where the AI responds in different personas using Google's Gemini API |
| 3 | [`memory-vault-chatbot`](./memory-vault-chatbot) | A stateful Streamlit chatbot that remembers conversation history across reruns using `st.session_state` |
| 4 | [`ai-image-studio`](./Upgrading%20the%20AI%20Image%20Studio) | A Streamlit AI image generator using the Pollinations API, with working width/height controls, dynamic downloadable file names, a "Magic Enhance" prompt booster, and a "Surprise Me" random prompt generator |

---

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- Pollinations API
